# Lumina Blog

一个现代风格的个人博客网站，使用 Vue 3 + TypeScript 构建。

## 功能特性

- 🎨 现代霓虹风格设计，支持明暗主题切换
- 📱 响应式布局，适配各种设备
- 🔍 文章搜索和分类筛选
- 🏷️ 标签筛选系统
- ✨ 流畅的动画和过渡效果
- 🧪 完整的单元测试

## 技术栈

- **框架**: Vue 3 (Composition API)
- **语言**: TypeScript
- **构建工具**: Vite
- **路由**: Vue Router 4
- **状态管理**: Pinia
- **测试**: Vitest + Vue Test Utils
- **代码规范**: ESLint

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 运行测试
npm run test

# 代码检查
npm run lint
```

## 项目结构

```
src/
├── components/     # 组件
├── composables/    # 组合式函数
├── data/          # 模拟数据
├── router/        # 路由配置
├── stores/        # Pinia 状态管理
├── styles/        # 全局样式
├── types/         # TypeScript 类型
├── views/         # 页面视图
├── App.vue        # 根组件
└── main.ts        # 入口文件
```
