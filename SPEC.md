# 个人博客网站规格文档

## 1. 项目概述

- **项目名称**: Lumina Blog
- **项目类型**: 单页面博客网站 (SPA)
- **核心功能**: 展示个人博客文章、支持文章分类、标签筛选、文章详情阅读
- **目标用户**: 技术博主、开发者社区

## 2. UI/UX 规格

### 布局结构

- **Header**: 固定顶部导航，包含 Logo、导航链接、主题切换按钮
- **Hero**: 首屏大图区域，带有动态粒子/光效背景
- **文章列表**: 卡片式布局，支持无限滚动
- **文章详情**: 全宽布局，带有目录导航
- **Footer**: 简洁底部，包含社交链接和版权信息

### 响应式断点
- Mobile: < 768px (单列布局)
- Tablet: 768px - 1024px (双列布局)
- Desktop: > 1024px (三列布局)

### 视觉设计

#### 配色方案 (暗色主题 - 默认)
- **主色**: #00D9FF (霓虹青色)
- **次色**: #7B61FF (电子紫)
- **强调色**: #FF6B9D (霓虹粉)
- **背景深**: #0A0E17 (深空黑)
- **背景浅**: #141B2D (星云蓝黑)
- **文字主**: #E8EAED (月光白)
- **文字次**: #8B95A5 (银灰)

#### 配色方案 (亮色主题)
- **主色**: #0891B2 (海洋青)
- **次色**: #6366F1 (靛蓝)
- **背景**: #F8FAFC (云白)
- **背景浅**: #F1F5F9 (雾灰)
- **文字主**: #1E293B (墨炭)
- **文字次**: #64748B (石板灰)

#### 字体
- **标题**: "Outfit", sans-serif (Google Fonts)
- **正文**: "Source Sans 3", sans-serif (Google Fonts)
- **代码**: "JetBrains Mono", monospace (Google Fonts)

#### 间距系统
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px
- 3xl: 64px

#### 视觉效果
- **卡片**: 玻璃拟态效果 (backdrop-filter: blur)
- **发光**: CSS box-shadow 配合主色发光
- **动画**: 
  - 页面切换: fade-slide 过渡
  - 卡片悬停: 上浮 + 发光增强
  - 按钮: 脉冲发光效果

### 组件列表

1. **NavBar**: 顶部导航栏
2. **ThemeToggle**: 主题切换开关
3. **HeroSection**: 首屏展示区
4. **ArticleCard**: 文章卡片
5. **ArticleList**: 文章列表容器
6. **ArticleDetail**: 文章详情页
7. **TagFilter**: 标签筛选器
8. **CategoryTabs**: 分类标签页
9. **SearchBar**: 搜索框
10. **Footer**: 页脚

## 3. 功能规格

### 核心功能
1. **文章列表展示**: 展示所有博客文章，支持分页/无限滚动
2. **文章分类**: 按分类筛选文章 (前端、后端、全栈、其他)
3. **标签筛选**: 多标签组合筛选
4. **文章搜索**: 按标题/内容搜索
5. **主题切换**: 明暗主题切换，支持系统偏好和本地存储
6. **文章详情**: 阅读完整文章内容
7. **目录导航**: 文章内快速跳转

### 数据模拟
- 使用本地 JSON 数据模拟文章内容
- 至少 6 篇示例文章

### 用户交互
- 点击卡片跳转到文章详情
- 点击标签筛选文章
- 主题切换带过渡动画

## 4. 工程化规范

### 技术栈
- **框架**: Vue 3 (Composition API)
- **语言**: TypeScript
- **构建工具**: Vite
- **路由**: Vue Router 4
- **状态管理**: Pinia
- **测试**: Vitest + Vue Test Utils
- **代码规范**: ESLint + Prettier
- **CSS**: Scoped CSS + CSS Variables

### 目录结构
```
src/
├── assets/          # 静态资源
├── components/      # 组件
├── composables/     # 组合式函数
├── router/          # 路由配置
├── stores/         # Pinia stores
├── types/          # TypeScript 类型
├── views/          # 页面视图
├── data/           # 模拟数据
├── App.vue
└── main.ts
```

## 5. 测试用例要求

### 单元测试
- 组件渲染测试
- 组件交互测试 (点击、输入等)
- 路由跳转测试
- 状态管理测试

1. ✅ 项目使用 Vue3 + TypeScript + Vite 构建
2. ✅ 包含至少 6 篇示例文章
3. ✅ 支持明暗主题切换
4. ✅ 支持文章分类和标签筛选
5. ✅ 响应式布局适配移动端
6. ✅ 具备动画和过渡效果
7. ✅ 包含单元测试用例
